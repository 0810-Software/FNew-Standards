# FNew Standards ver. 20.4.30.2 document

## Tag

```HTML
<!--
FNew Standards ver. 20.4.30.2 apply to this file.
Visit https://0810-software.github.io/FNew-Standards/20.4.30.2 for more information.

Tagger: {(@user)name of tagger}

Contributors:
- {(@user)names of everyone who contributed to this file}
-
Please list your name above when comitting to this file for the first time.
-->
```

This tag may only be added to files if they comply with the rules specified in this file. The {}-variables have to be replaced by their values before tagging.

This tag should be placed between the `<HTML>` and the `<HEAD>`-element.

## Requirements

### Global

*Global means everywhere where no other FNew Standard specifications are specified*

#### HTML

The `<HTML>`-element should have the following attributes:

```HTML
<HTML lang="en">
    ...
</HTML>
```

### Posts

#### Title

The title of a post must always be:

````html
<title>FNew - Post: {post name}</title>
````

#### Keywords

The post title and post date should be in the keywords.

```HTML
<META NAME="keywords" LANG="en" CONTENT="{post title}, {post date}, FNew, Social ...
```

### Content

Content that differs per-page should also follow some guidelines.

#### Images

In every single `<IMG>`-element there should be a value `loading="lazy"`.

Example:

````HTML
<img loading="lazy" src="https://software0810.files.wordpress.com/2019/11/cropped-0810software-2.png?w=32" alt="Go to the 0810 Software site@wordpress.com" height="32"> 
````
